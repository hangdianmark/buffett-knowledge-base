# 巴菲特投资智慧库

一个完整的、离线的沃伦·巴菲特投资知识库，供终身学习使用。

## 内容概览

| 类别 | 数量 | 说明 |
|------|------|------|
| 📌 核心概念 | 35个 | 价值投资、护城河、内在价值等关键概念 |
| 🏢 投资公司 | 61家 | 可口可乐、苹果、美国运通等投资分析 |
| 📝 股东信件 | 60封 | 伯克希尔1965-2024年完整股东信 |
| 🤝 合伙人信 | 17封 | 1956-1964年早期投资理念 |
| 👤 关键人物 | 7位 | 查理·芒格、本杰明·格雷厄姆等 |
| ⭐ 特别信件 | 4封 | 危机时期的重要投资建议 |

**总计：186个知识页面**

## 使用方法

1. **解压缩**：解压下载的ZIP文件到任意目录
2. **打开网站**：进入解压后的文件夹，双击 `index.html` 文件
3. **开始学习**：使用浏览器打开后即可开始浏览和搜索
